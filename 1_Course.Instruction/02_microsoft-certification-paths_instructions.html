Associate level

The Azure credentials at the associate level include Azure Administrator, Azure Developer Azure AI Engineer, Azure Data Scientist, Azure Data Engineer, and Azure Security Engineer.

Expert level

The expert level currently includes Azure DevOps Engineer and Azure Solutions Architect.
